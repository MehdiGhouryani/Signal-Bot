# راهنمای اجرای اولین بار

## ۱. پیش‌نیازها

```bash
python3 --version   # باید ۳.۱۰ یا بالاتر باشه
pip install -r requirements.txt
```

## ۲. تنظیمات (.env)

```bash
cp .env.example .env
```

بعد `.env` رو باز کن و حداقل این دوتا رو پر کن (بقیه اختیاری‌ان):

- `BOT_TOKEN` — از [@BotFather](https://t.me/BotFather) بگیر.
- `ADMIN_IDS` — آیدی عددی خودت (با کاما جدا اگه چندتا ادمین داری).

برای اینکه آیدی عددی خودت رو بفهمی، به [@userinfobot](https://t.me/userinfobot) پیام بده.

بقیه‌ی مقادیر (`NOWPAYMENTS_API_KEY`, `CHANNEL_ID`, وب‌هوک IPN) اختیاری‌ان — اگه
خالی بذاریشون، همون بخش‌ها (پرداخت آنلاین، پست خودکار کانال) غیرفعال می‌مونن
بدون اینکه ربات کرش کنه.

## ۳. اجرا

```bash
python main.py
```

اگه همه‌چیز درست باشه، این رو می‌بینی:

```
Signal Master Bot — فاز ۵ ✅
...
```

و ربات آماده‌ست تو تلگرام باهاش صحبت کنی.

## ۴. چک‌لیست تست دستی (پیشنهادی، قبل از اعلام عمومی به کاربرها)

این‌ها رو با خودت (به‌عنوان ادمین) و در صورت امکان یه اکانت تست دیگه (به‌عنوان
کاربر عادی) امتحان کن:

- [ ] `/start` — منو با اطلاعات درست (رول، سطح) میاد
- [ ] ثبت Full Signal با عکس، و یه بار هم فقط با متن (بدون عکس)
- [ ] ثبت Fast Call با جهت، و یه بار با «رد کردن جهت»
- [ ] تو یه گروه: `/fastcall COIN long` و `/fullsignal` (با و بدون عکس)
- [ ] تأیید/رد یه سیگنال از پنل ادمین؛ اگه VIP Helper داری، با اکانت اون هم امتحان کن
- [ ] ثبت نتیجه‌ی یه سیگنال (هر ۵ گزینه رو حداقل یک‌بار امتحان کن)
- [ ] لیدربورد، پروفایل، تابلوی افتخار، فید عمومی سیگنال‌های فعال
- [ ] واریز به استخر جایزه (مبلغ ثابت و دلخواه) — چون NOWPayments واقعی لازمه،
      اگه کلیدش رو نداری فعلاً رد کن
- [ ] حمایت مستقیم از یه تحلیلگر از روی فید عمومی
- [ ] پنل ادمین: مدیریت کاربران (بلاک/امتیاز/رول/VIP/پاداش)، خروجی اکسل، پیام همگانی
- [ ] «🏆 پایان دوره» — مطمئن شو فصل جدید خودکار شروع می‌شه (رفع باگ اخیر)

## ⚠️ چیزهایی که هنوز نیاز به تست واقعی (نه mock من) دارن

این‌ها رو من با شبیه‌سازی تست کردم، ولی چون سندباکسم به تلگرام/NOWPayments
دسترسی نداره، **جایگزین تست واقعی نیستن**:

1. **پرداخت واقعی NOWPayments** — ساخت فاکتور، پرداخت واقعی، و چک وضعیت (چه
   دستی چه با وب‌هوک اگه فعالش کردی).
2. **وب‌هوک IPN** — اگه `NOWPAYMENTS_IPN_SECRET`/`IPN_CALLBACK_URL` رو پر
   کردی، حتماً با یه پرداخت واقعی کوچیک تستش کن.
3. **حجم بالای پیام در گروه‌های شلوغ** — رفتار ربات با محدودیت‌های تلگرام
   (flood control) در استفاده‌ی سنگین.
